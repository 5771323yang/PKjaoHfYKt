# 垃圾检测与分类系统 python1005

## 项目概述

本项目是基于深度学习的垃圾检测与分类系统，包括用户界面（UI）、使用YOLOv8的Python代码以及相应的数据集。

## 技术栈

- **前端**：UI界面
- **后端**：Python
- **深度学习框架**：YOLOv8

## 功能模块

- 垃圾检测
- 垃圾分类
- 数据集管理

## 系统角色

- 用户：通过UI界面进行垃圾图片的上传、检测结果查看

## 运行环境

- 操作系统：兼容Windows、Linux、macOS
- 硬件要求：具备一定GPU性能以支持深度学习模型运行
- 软件依赖：Python及相关库

## 部署步骤

1. 安装Python环境
2. 克隆项目代码
3. 准备数据集
4. 运行项目

## 目录结构

```
.
├── dataset    // 数据集目录
├── models     // 模型文件目录
├── src        // 源代码目录
│   ├── ui     // UI界面代码
│   └── main.py // 主程序
└── README.md  // 项目说明文档
```

## 核心亮点

- **高性能**：基于YOLOv8的高效深度学习模型
- **易用性**：配有图形用户界面，便于非技术用户操作
- **扩展性**：支持各类自定义数据集，便于模型训练与优化
